# 參與貢獻指南 Contributing Guide

感謝您有興趣為紫微斗數開源知識庫做出貢獻！

## 🌟 如何參與

### 內容貢獻
- **文章撰寫**：分享您的紫微斗數學習心得與研究
- **術語補充**：補充專業術語解釋
- **勘誤修正**：協助修正內容中的錯誤

### 技術貢獻
- **排盤工具改進**：提交 Pull Request 優化排盤邏輯
- **AI 解盤優化**：改善 AI 解盤的準確性與表達

## 📝 提交規範

1. Fork 本專案
2. 創建功能分支：`git checkout -b feature/your-feature`
3. 提交更改：`git commit -m 'Add: 新增某某內容'`
4. 推送分支：`git push origin feature/your-feature`
5. 發起 Pull Request
