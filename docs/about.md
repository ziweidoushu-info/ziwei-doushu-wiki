# 關於紫微星盤網

[紫微星盤網](https://ziweidoushu.info)（ZiWeiDouShu.info）是專業的**紫微斗數在線服務平台**，致力於推廣傳統紫微斗數命理文化。

## 🎯 我們的服務

### 免費在線排盤
輸入國曆（公曆）出生日期和時間，即時生成完整紫微斗數命盤，包含十四主星、六吉六煞、四化飛星等所有星曜資訊。

### AI 智能解盤
運用人工智能技術，提供專業級的命盤解讀服務，涵蓋性格分析、事業發展、感情運勢、財運走向等多個面向。

### 專業命理文章
收錄超過 **1,400 篇**深度命理文章，從入門基礎到進階技法，建立完整的紫微斗數知識體系：
- 紫微斗數基礎入門
- 十二宮位詳細解析
- 十四主星性格特質
- 四化飛星技法運用
- 大限流年運勢推算
- 格局判斷與實戰案例

### 術語字典
收錄紫微斗數專業術語，方便初學者快速查詢理解。
