# 术语卡片：天相

- 分类：主星
- 原词条：[https://ziweidoushu.info/dict/zhuxing/tianxiang](https://ziweidoushu.info/dict/zhuxing/tianxiang)

## 摘要节选

<div class="article-content">
	<div class="introduction-block">
		<p>您身邊有沒有這樣一種人？他們是天生的「和事佬」，是團隊裡的「潤滑劑」，是那個總在說「大家別吵，聽我說兩句」的老好人。他們正直、穩重、富有同情心，但似乎也總在替人背鍋，常常陷入兩難，感覺自己被夾在中間，裡外不是人。</p>
		<p>如果您認識這樣的人，或者您自己就是，那麼您很可能已經見識到了「天相星」最真實、最鮮活的模樣。</p>
	</div>
	<div class="main-content">
		<h2 class="chapter-title">一、核心定義 (What is it?)</h2>
		<p>天相星，在紫微斗數中被稱為<span class="highlight-text">「印星」</span>，五行屬水，是中天星曜中的一顆重要輔佐之星。它象徵著<span class="highlight-text">規則、契約、公正與服務</span>，是命盤中代表<span class="highlight-text">秩序與信譽</span>的樞紐型星曜。</p>
		<h2 class="chapter-title">二、關鍵象徵 (Keywords)</h2>
		<p>為了快速理解天相，您可以記住以下幾個關鍵詞：</p>
		<ul>
			<li><span class="highlight-text">宰相/高級幕僚：</span> 它不是發號施令的皇帝，而是輔佐君王、執行政令的百官之長。</li>
			<li><span class="highlight-text">行走的公章：</span> 它的核心價值在於「認證」與「擔保」，代表著信用、合同與文書。</li>
			<li><span class="highlight-text">好好先生：</span> 天性善良，樂於助人，有服務精神，但也容易因為心軟而陷入人情困境。</li>
			<li><span class="highlight-text">原則與秩序：</span> 骨子裡有強烈的秩序感，注重規矩、流程和公平正義。</li>
			<li><span class="highlight-text">中間人/調停者：</span> 善於在矛盾雙方中斡旋，是天生的外交官和談判者。</li>
			<li><span class="highlight-text">受環境影響：</span> 這是天相最關鍵的特質，它極度容易被左右相鄰的星曜「定義」，俗稱「近朱者赤，近墨者黑」。</li>
			<li><span class="highlight-text">衣食之星：</span> 天相坐命的人通常很注重儀表和生活品味，對吃穿都有一定的要求。</li>
		</ul>
