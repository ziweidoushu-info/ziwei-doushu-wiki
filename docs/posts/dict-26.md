# 术语卡片：破軍

- 分类：主星
- 原词条：[https://ziweidoushu.info/dict/zhuxing/pojun](https://ziweidoushu.info/dict/zhuxing/pojun)

## 摘要节选

<div class="article-content">
    <div class="introduction-block">
        <p>很多人一聽到「破軍」這兩個字，腦子裏冒出的大概率是戰爭、破壞、六親無靠……彷彿天生就是來搞事情的。就像我們提起喬布斯，總會想起他那句著名的「現實扭曲力場」，他以一己之力，把諾基亞和摩托羅拉辛苦建立的手機帝國砸了個稀巴爛。但奇怪的是，我們最終記住的，不是那個「破壞者」喬布斯，而是創造了iPhone的「開創者」喬布斯。</p>
        <div class="emphasis-box">
            <p>這，就是理解破軍的第一把鑰匙。</p>
        </div>
    </div>
    <div class="main-content">
        <h2 class="chapter-title">一、核心定義 (What is it?)</h2>
        <p><strong>破軍</strong>是一顆主動求變的<strong class="highlight-text">先鋒星曜</strong>，其核心驅動力是通過<strong class="highlight-text">消耗、破壞</strong>現有秩序與框架，來<strong class="highlight-text">開創</strong>一個全新的局面。它是紫微斗數「殺破狼」格局中的關鍵一環，象徵著「不破不立」的變革力量。</p>
        <h2 class="chapter-title">二、關鍵象徵 (Keywords)</h2>
        <p>為了快速理解破軍，您可以記住以下幾個關鍵詞：</p>
        <ul>
            <li><strong>先鋒大將：</strong> 永遠衝在第一線的急先鋒，負責攻城拔寨，打破僵局。</li>
            <li><strong>推倒重來：</strong> 它的字典裏沒有「修修補補」，只有「格式化重裝」。</li>
            <li><strong>消耗與開創：</strong> 一體兩面的核心特質，先有巨大的能量消耗，後有全新的局面誕生。</li>
            <li><strong>破舊立新：</strong> 這是它的使命，也是它的宿命。</li>
            <li><strong>極致體驗官：</strong> 凡事都要親力親為，追求極致的、前所未有的體驗，無論是好是壞。</li>
            <li><strong>「我不要你覺得」：</strong> 主觀意識極強，以自我感受為第一標準，帶有強烈的情緒驅動色彩。</li>
        </ul>
