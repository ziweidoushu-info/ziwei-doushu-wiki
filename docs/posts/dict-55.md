# 术语卡片：三合

- 分类：技法
- 原词条：[https://ziweidoushu.info/dict/jifa/sanhe](https://ziweidoushu.info/dict/jifa/sanhe)

## 摘要节选

<div class="article-content">
<div class="introduction-block">
<p>很多人都聽過一個詞，「鐵三角」，說的是三方力量穩固，配合默契。在紫微斗數裡，也有這麼一個「鐵三角」，它幾乎決定了你人生這部大戲的主線劇情是喜劇、悲劇，還是懸疑劇。但最有意思的是，有時候，一個看似完美的「黃金鐵三角」，在現實中卻可能活成一個「青銅局」。這是為什麼？我們今天就來聊聊命盤裡最核心的聯動機制——<span class="highlight-text">三合</span>。</p>
</div>
<div class="main-content">
<h2 class="chapter-title">一、核心定義 (What is it?)</h2>
<p>在紫微斗數命盤中，<span class="highlight-text">三合</span>指的是以特定宮位為基準，與其呈120度角關係、共同構成正三角形格局的另外兩個宮位，它們共同組成了該領域的核心能力架構與長期資源支持系統。</p>
<h2 class="chapter-title">二、關鍵象徵 (Keywords)</h2>
<p>為了快速理解<span class="highlight-text">三合</span>，您可以記住以下幾個關鍵詞：</p>
<ul>
<li><span class="highlight-text">天生的盟友：</span> 這是你一出場就自帶的隊友，他們跟你目標一致，榮辱與共。</li>
<li><span class="highlight-text">鐵三角：</span> 它們共同構成了一個穩定結構，決定了你處理某一領域事務的基本盤和風格。</li>
<li><span class="highlight-text">資源包：</span> 你的天賦、能力、價值觀這些「出廠設置」，都裝在這個三合的資源包裡。</li>
<li><span class="highlight-text">底層操作系統：</span> 如果說命宮是CPU，那三合宮位就是跟它無縫協作的內存和顯卡，共同決定了這台「電腦」的性能上限。</li>
<li><span class="highlight-text">長期主義：</span> 三合的影響是深遠且持久的，它不像流年那樣來得快去得也快，而是貫穿一生的背景音樂。</li>
<li><span class="highlight-text">協同效應：</span> 三個宮位裡的星曜是1+1+1>3還是&lt;3，看的正是這種化學反應。</li>
<li><span class="highlight-text">舒適區：</span> 一個過於和諧的三合，有時也會成為讓你不願走出去的「溫柔鄉」。</li>
</ul>
<h2 class="chapter-title">三、形象比喻 (Analogy)</h2>
<p>要徹底搞懂「三合」，最貼切的比喻，就是把它想像成一個<span class="highlight-text">「創業合夥人團隊」</span>。</p>
